## About the Project

TEXP is a modern, interactive text animation generator that empowers designers and developers to create stunning GSAP animations without writing code. With an intuitive interface and real-time preview, craft pixel-perfect text animations in seconds.

**Key Highlights:**
- Real-time animation preview
- Comprehensive animation controls (scale, rotation, opacity, blur, and more)
- Advanced styling options with custom CSS
- Gradient and image backgrounds
- Split text animation with stagger effects
- Fully responsive design
- Dark mode support
- Export animations in multiple frameworks (Vanilla JS, React, Vue)
- Support for both JavaScript and TypeScript

## Features

### Animation Engine
- **Tween Types:** To, From, FromTo animations
- **Easing Functions:** All GSAP easing options
- **Advanced Controls:** Duration, delay, repeat, yoyo, stagger
- **Effects:** Blur filter, transform properties, opacity control

### Styling
- **Typography Control:** Font family, size, weight, letter spacing
- **Text Styling:** Text transform, decoration, line height
- **Visual Effects:** Background colors, gradients, custom overflows
- **Responsive:** Mobile-first design

### Export Options
- Generate production-ready code
- Multiple framework support
- TypeScript ready

## Getting Started

### Prerequisites
- Node.js 18+
- pnpm (recommended) or npm

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/texp.git

# Navigate to project
cd gsap-playground

# Install dependencies
pnpm install

# Start development server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Usage

1. **Enter Text:** Input the text you want to animate
2. **Configure Animation:** Use the animation panel to set properties
3. **Style:** Apply custom CSS and background effects
4. **Preview:** See real-time animation in the canvas
5. **Export:** Generate code for your project
